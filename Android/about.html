ABOUT


  1. This is very simple deep learning application which can detect objects in the image with top probabilities asscociated with it using vgg16 weights.
  2. I extended further to 6 diseases which are Brain tumor, Malaria, Optical Coherence Tomography, Diabetes Retinopathy, Breast Cancer and Pneumonia using chest x-ray images
  3. It best works for single object in a image.

FUTURE WORK


  1. Planning to build a heart beat detector using a CNN-RNN architecture and integrate it with this application.
  2. The future updates will also see on how to detect and interpret ECG signals.
  3. Will keep building this project and add more diseases as I move along. Our primary focus for now is to improve the brain tumor model with much more data samples.
  4. I will implement GPS based hospital recommendation system using the Google Maps API. I are not yet finished with the whole code.
  5. I have plans to train several other models on medical datasets and add more useful features to this app. For example, I tried implementing the google maps api for nearest hospital suggestions this time but it hasn’t been fully implemented. I will keep them for future updates.
  6. I have planned to build a separate app for deploying the YOLO model using openCV, so that you can use it for a wide range of things — from having fun to having video surveillance.
  7. In this application, I will try to improve the accuracy of the base models further.
  8. I have planned to write this in angular in future so that it becomes much more flexible and you get a faster and cleaner UI.
